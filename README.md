Startup War Simulator

Requirements:
- Node 18+
- Groq API key

Setup:

1. Clone or paste project into folder
2. npm install
3. Add .env.local
4. npm run dev

Production:
npm run build
npm start

Groq:
Get API key from console.groq.com
Add to .env.local

Notes:
- In-memory state
- No database
- Optimized compressed state format
- Hybrid rule engine
